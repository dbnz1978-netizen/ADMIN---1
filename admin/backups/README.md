# Директория резервных копий

Эта директория используется для хранения резервных копий сайта.

## Безопасность

- Директория защищена от прямого доступа через веб-сервер (.htaccess)
- Доступ к резервным копиям только через административную панель
- Только администраторы могут скачивать и удалять резервные копии

## Обслуживание

Рекомендуется регулярно очищать старые резервные копии для экономии дискового пространства.
